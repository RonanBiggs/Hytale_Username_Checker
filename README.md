# Hytale_Username_Checker
Check availability of hytale usernames

To run you will need a list of usernames formatted with one per line.   
Your Hytale token from the url "https://accounts.hytale.com/profiles?token=[YOUR-TOKEN]"  
And the value from the "ory_kratos_session" cookie. To get this go to the hytale url referenced above -> ctrl+shift+i -> storage -> copy paste the value
